What is libmng-py?

libmng-py is a Python wrapper around the libmng C library. It currently supports the most important parts of the library and supports displaying and usage with the most popular toolkits (such as pygame and wxPython).

I started it because I need an animation format with full alpha channel support (as appose to GIF which only has a alpha mask) and no other python library supported MNG.


Goals

  • full support for all libmng functionality
  • out of the box support for popular toolkits

Why python?

Python is cross-platform and object-oriented. It's easy-to-learn, easy-to-use, easy-to-read, easy-to-extend. It is also very important that python is increasingly popular.
